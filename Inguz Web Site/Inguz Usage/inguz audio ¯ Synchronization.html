Synchronization

SlimServer can synchronize playing between more than one Squeezebox or Transporter.  This is really great for whole-house audio systems.  Unfortunately, it doesn’t work quite perfectly with the InguzDSP system.

All synchronized Squeezeboxes are processed through one “pipeline”.

There’s no easy way to change this.
